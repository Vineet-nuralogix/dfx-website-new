DeepAffex® Cloud API

The DFX Cloud API enables customers to communicate with our DFX cloud service. It allows them to programmatically access our DFX cloud service and manage their organization's DeepAffex® account data and results.

The DFX Cloud API is published as a reference specification (documentation) describing the programmatic interfaces (or endpoints) that are provided by the DeepAffex® cloud engine.

These network endpoints allow a remote client (program) to invoke and access DeepAffex® micro-services and features.

The DFX Cloud API allows the use of REST for synchronous (blocking) transactions as well as Web-socket protocols for asynchronous (non-blocking) communications.
